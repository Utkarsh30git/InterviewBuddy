# 🎤 InterviewBuddy AI

An AI-powered mock interview platform that helps users prepare for technical and HR interviews through intelligent question generation, resume analysis, and real-time AI feedback.

---

## 🚀 Features

- 🔐 Google Authentication (Firebase)
- 📄 AI Resume Analysis
- 🤖 AI-Powered Interview Question Generation
- 💬 Technical, HR & Behavioral Interview Modes
- 📚 Interview History
- 💳 Credit-Based Interview System
- 📊 Performance Report
- ☁️ MongoDB Atlas Integration
- ⚡ OpenRouter LLM Integration

---

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- Redux Toolkit
- Axios
- Firebase Authentication

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

### AI
- OpenRouter API
- LLM-based Resume Analysis
- AI Interview Question Generation

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Utkarsh30git/InterviewBuddy.git
```

### Install Frontend

```bash
cd client
npm install
```

### Install Backend

```bash
cd ../server
npm install
```
